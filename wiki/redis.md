# Redis

### 备忘

* 实例(1)
  * 端口: 6379
  * 安装目录: /opt/redis/
  * 数据目录: /opt-data/redis-6379/
  * 配置文件目录: /opt/redis/etc/redis-6379.conf
  * 日志输出: 无
  * PID文件: /var/run/redis-6379.pid
  * 注册服务: redis-6379.service
